# Juno User Manual

**Juno** is a lightweight, always-on-top floating quick-notes application designed to keep your thoughts organized without interrupting your workflow.

## üöÄ Getting Started

### Installation
1.  Run the `Juno Setup 1.0.0.exe` installer.
2.  Follow the prompts to install Juno on your computer.
3.  Once installed, Juno will launch automatically.

### The Floating Ball
*   **Always on Top**: Juno appears as a small, unobtrusive floating ball on your screen.
*   **Move It**: Click and drag the ball to any position on your screen. It will stay there!
*   **Open Notes**: Click the ball (or press `Ctrl+Shift+N`) to expand the full notes widget.

### üìù Quick Notes
*   **Create**: Type your note in the input box and press "Add Note".
*   **Colors**: Choose from 5 color codes (Blue, Red, Green, Yellow, White) to categorize your notes.
*   **Edit/Delete**: Hover over any note to see options to Edit (pencil icon) or Delete (trash icon).

### ‚è∞ Reminders
Never forget a task! You can set a one-time reminder for any note.
1.  Click the **Calendar** icon to set a date.
2.  Click the **Clock** icon to set a time.
3.  When the time comes, Juno will send a system notification. Clicking the notification opens the app.

### üé® Themes
*   **Dark/Light Mode**: Open the **Settings** (gear icon) inside the widget to toggle between Dark and Light themes.

*   **`Ctrl + Shift + N`**: Toggle the Juno window (Show/Hide) instantly from anywhere.

Juno runs in the background. Check your Windows System Tray (near the clock) for the Juno icon:
*   **Right-Click**: Access the menu to "Toggle Expand" or "Quit" the app.
*   **Double-Click**: Quickly show or hide the floating ball.

## ‚ùì Troubleshooting

**Q: The app disappeared!**
A: Check the System Tray (^ arrow near your clock). If the icon is there, double-click it. If not, relaunch Juno from the Start Menu.

**Q: I can't drag the window.**
A: You can only drag the **collapsed ball**. If the widget is expanded (showing notes), collapse it first by clicking the "Minimize" (-) button or pressing `Esc`.

**Q: Notifications aren't showing.**
A: Ensure your Windows "Focus Assist" or "Do Not Disturb" mode is off. Juno creates standard Windows notifications.
